### Multi layer perceptron  
Implements a character level multi layer perceptron based on [Bengio et al. 2003 MLP language model paper](https://www.jmlr.org/papers/volume3/bengio03a/bengio03a.pdf)  
  
Inspired by Andrej Karpathy's first makemore video:
